## 🎯 实际效果示例

### 场景 1：技术术语学习（idempotent）

**v1 算法**：
```
初始 easeFactor: 2.5
学习路径: 0天 → 1天 → 6天 → 15天 → ...
```

**v2 算法**：
```
初始 easeFactor: 2.0 (自动识别为技术词)
学习路径: 10分钟 → 1天 → 3天 → 6天 → 12天 → ...
```

**改进**：
- 初始难度更合理（2.0 vs 2.5）
- 前期巩固更频繁（10分钟 + 3天 vs 6天）
- 总体增长更保守（easeFactor 2.0 vs 2.5）

### 场景 2：遗忘后恢复

**v1 算法**：
```
学到 30 天间隔 → 遗忘 → 重置到 1 天
用户感受：前功尽弃，挫败感强
```

**v2 算法**：
```
学到 30 天间隔 → 遗忘 → 降到 8 天 (30 * 0.25)
用户感受：保留部分进度，更有信心
```

### 场景 3：困难词循环

**v1 算法**：
```
学习 → 遗忘 → 1天 → 遗忘 → 1天 → 遗忘 → 1天 ...
easeFactor 不断下降，但增长速度不变
```

**v2 算法**：
```
学习 → 遗忘 (lapses=1) → 遗忘 (lapses=2) → 遗忘 (lapses=3)
→ 系统识别为困难词，easeFactor 上限降至 2.0
→ 后续增长更保守，减少频繁遗忘
```
